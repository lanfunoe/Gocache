## Notes

- All API responses are accessed through `response.data` due to the axios interceptor in `src/utils/request.js` which automatically returns `response.data`.
- Some endpoints only use the response for status validation and don't access specific data fields.
- Array responses like `/album/detail` return data in `response.data[0]` format.
- Field access patterns like `response.data.songs[].hash` indicate that the `hash` field is accessed from each item in the `songs` array.
- Multiple field paths on the same line (e.g., for `/search`) indicate that different fields are used depending on the search type parameter.
